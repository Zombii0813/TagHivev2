# TagHive 使用指南

## 快速入门

### 1. 首次启动

启动 TagHive 后，首先需要选择一个工作区（文件夹）：

1. 点击主界面中的"选择文件夹"按钮
2. 在弹出的对话框中浏览到目标文件夹
3. 点击"选择文件夹"确认

### 2. 扫描文件

选择工作区后，TagHive 会自动开始扫描文件：

- 扫描进度会显示在界面上
- 扫描完成后，文件列表会自动显示
- 可以随时点击设置中的"重新扫描"来更新索引

### 3. 添加标签

#### 创建新标签

1. 在左侧标签面板中点击"新建"按钮
2. 输入标签名称
3. 选择标签颜色（可选）
4. 点击"确定"

#### 为文件添加标签

**方式一：在文件详情面板中**
1. 选中一个或多个文件
2. 在右侧详情面板中点击"编辑"标签
3. 勾选要添加的标签
4. 点击"保存"

**方式二：批量操作**
1. 选中多个文件（Ctrl+点击或框选）
2. 右键选择"批量添加标签"
3. 选择要添加的标签

### 4. 搜索文件

#### 文本搜索

在顶部搜索框中输入文件名或关键词：
- 支持模糊匹配
- 支持全文搜索
- 搜索结果实时更新

#### 按类型筛选

点击搜索框旁边的"类型"下拉菜单：
- 全部
- 图片 (jpg, png, gif, etc.)
- 视频 (mp4, mov, mkv, etc.)
- 文档 (pdf, doc, xls, etc.)
- 音频 (mp3, wav, flac, etc.)

#### 按标签筛选

点击左侧标签列表中的标签：
- 单击：筛选该标签的文件
- Ctrl+单击：多选标签
- 再次点击：取消筛选

### 5. 查看文件详情

选中文件后，右侧详情面板会显示：

- **预览**: 图片/视频预览（如支持）
- **基本信息**: 文件名、类型、大小、路径
- **修改时间**: 最后修改日期
- **标签**: 已添加的标签列表
- **操作**: 打开文件、打开文件夹等

### 6. 视图模式

在工具栏中可以切换视图模式：

- **网格模式**: 缩略图+文件名，适合图片浏览
- **列表模式**: 详细信息，适合文件管理

### 7. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + A | 全选 |
| Ctrl + 点击 | 多选 |
| Shift + 点击 | 范围选择 |
| Delete | 删除选中文件记录 |
| Ctrl + F | 聚焦搜索框 |
| Ctrl + 1/2 | 切换网格/列表视图 |

## 高级功能

### 1. 文件监控

TagHive 会自动监控工作区的文件变化：
- 新增文件自动添加到索引
- 修改的文件自动更新
- 删除的文件自动从索引移除

### 2. 缩略图缓存

- 图片和视频会自动生成缩略图
- 缩略图缓存到本地，加速后续浏览
- 可以手动清理缓存（设置中）

### 3. 数据备份

TagHive 的数据库文件位于：
- Windows: `%USERPROFILE%\.taghive\taghive.db`
- macOS/Linux: `~/.taghive/taghive.db`

建议定期备份此文件以保护标签数据。

## 故障排除

### 扫描无结果

1. 确认工作区路径正确
2. 检查文件夹访问权限
3. 尝试重新扫描

### 标签不显示

1. 确认已创建标签
2. 检查是否为文件正确添加了标签
3. 尝试刷新页面

### 性能问题

1. 减少同时显示的文件数量
2. 使用搜索和筛选缩小范围
3. 定期清理缩略图缓存

### 数据丢失

如果标签数据意外丢失：
1. 检查数据库文件是否存在
2. 从备份恢复数据库文件
3. 重新扫描工作区

## 隐私说明

- 所有数据存储在本地，不会上传到云端
- 文件索引仅包含元数据（文件名、大小等），不包含文件内容
- 缩略图缓存在本地生成和存储
